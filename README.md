React + Node.js Login Application

A simple full-stack Login Application built using:

⚛ Frontend: React (Vite)

🟢 Backend: Node.js + Express

🎨 Styling: Tailwind CSS

📌 Features

Login page with username & password

Backend API (POST /login) for authentication

Validates credentials (admin/admin)

Redirects to Welcome page on success

Displays error message on invalid credentials

Remembers username using localStorage

Proper HTTP status codes

Clean UI with Tailwind CSS

Rate limiting for login security
